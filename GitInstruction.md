## Работа с удаленными репозиториями.

### Что такое удаленный репозиторий и для чего он нужен

Удаленный репозиторий нужен для хранения файлов с контролем версий. Для совместной работы нескольких пользователей с файлами удаленно, каждый со своего рабочего места.
Подобный функционал часто используется для работы над open-source проектами. 
Чаще всего используется ресурс [GitHub](https://github.com/).
